Introduction and History

JavaScript is the most popular and widely used client-side scripting language
JavaScript is designed to add interactivity and dynamic effects to web pages.
JavaScript was originally developed as LiveScript by Netscape in the mid 1990s.
It was later renamed to JavaScript in 1995
  • JavaScript is officially maintained by ECMA (European Computer Manufacturers Association) as ECMAScript.

  • ECMAScript 6 (or ES6) is the latest major version of the ECMAScript standard.
